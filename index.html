Musical Keygen 0.0 More info

Familiarizing oneself with an instrument's note placement is important for any musician. This program allows the user to test their knowledge of the guitar fretboard.

The user is prompted with a random note that will generate at the set interval. To increase recognition difficulty one can add string number, chord/arpeggio, and fingering.

This tool can be used for any instrument or genre, but I built it to test myself on jazz guitar arpeggios.

Contact me with comments or requests at me@zacharysedefian.com.
