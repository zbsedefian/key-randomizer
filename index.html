Welcome.

This program helps the user test themselves on guitar fretboard knowledge. The user is prompted with a random note that will generate at the set interval. To increase difficulty one can add string number, chord (arpeggio), and fingering.

This tool can be used for any instrument or genre, but I built it to test myself on jazz guitar arpeggios.

Contact me with comments or requests at me@zacharysedefian.com.
